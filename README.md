# RISC OS Squash packer/unpacker
```
   squash c inputfile outputfile    compress to SQSH Squash file
   squash d inputfile outputfile    decompress SQSH Squash file
   squash cr inputfile outputfile   raw 13-bit Squash LZW compress
   squash dr inputfile outputfile   raw 13-bit Squash LZW decompress
```
